Consideraciones para la instalacion:

1) Antes que nada se debe tener instalado Wampserver ultima verson.
2) luego de descargado el archivo desde el repositorio Git, descomprimirlo.
3) Guardar la carpeta descomprimida en la carpeta "www" que se encuentra dentro de la carpeta principal "wamp".
4) Seguido se deben activar los servicios del servidor para finalmente poder visualizar el sitio de prueba.
5) La base de datos se encuentra en un archivo con extension .sql llamado "cafe".
6) luego de instalado la carpeta en el servidor, abren Phpmyadmin a crear la base de datos con el nombre cafe y luego se importa el archivo de base de datos o se puede copiar y pegar el codigo de base de datos en la pestaña SQL para una instalacion mucho mas rapida.
7) El sistema puede funcionar en una pc con caracsteristicas estándar.
